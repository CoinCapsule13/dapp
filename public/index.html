User Instructions

        

            
1. Introduction

            
Your app allows users to interact with the Coincapsule smart contract for managing their wallets, minting tokens, and performing transactions. Follow these steps to use the app effectively.

        

        

            
2. Getting Started

            
1. Access the App: Open your web browser and navigate to the URL where your app is hosted.

            
2. Connect Wallet: Click on the "Connect to MetaMask" button. Choose MetaMask from the available options and follow the prompts to connect your wallet.

        

        

            
3. Mint Tokens

            
1. Navigate to Minting Section: Find and click on the "Mint Tokens" section.

            
2. Enter Details: Input the number of tokens you wish to mint and provide a label for your new wallet.

            
3. Pay Minting Fee: Ensure you have enough ETH in your wallet to cover the minting fee (0.00005 ETH per token). The total fee will be calculated and displayed for confirmation.

            
4. Mint Tokens: Click the "Mint Tokens" button to initiate the minting process. Confirm the transaction in MetaMask and wait for the process to complete.

        

        

            
4. Get Token IDs

            
1. Enter Owner Address: Input the Ethereum address of the wallet you want to query.

            
2. Fetch Token IDs: Click the "Get Token IDs" button to retrieve and display the token IDs associated with the address.

        

        

            
5. Get Wallet Contents

            
1. Enter Token ID: Input the ID of the token whose contents you want to view.

            
2. Fetch Contents: Click the "Get Wallet Contents" button to display the contents of the specified wallet.

        

        

            
6. Freeze Wallet

            
1. Enter Token ID: Input the ID of the wallet you want to freeze.

            
2. Enter Password: Provide the password for authorization.

            
3. Freeze Wallet: Click the "Freeze Wallet" button to prevent further transactions from the wallet.

        

        

            
7. Unfreeze Wallet

            
1. Enter Token ID: Input the ID of the wallet you want to unfreeze.

            
2. Enter Password: Provide the password for authorization.

            
3. Unfreeze Wallet: Click the "Unfreeze Wallet" button to allow transactions from the wallet again.

        

        

            
8. Deposit ETH

            
1. Enter Token ID: Input the ID of the wallet to which you want to deposit ETH.

            
2. Enter ETH Amount: Provide the amount of ETH you want to deposit.

            
3. Deposit ETH: Click the "Deposit ETH" button to transfer ETH to the specified wallet.

        

        

            
9. Withdraw ETH

            
1. Enter Token ID: Input the ID of the wallet from which you want to withdraw ETH.

            
2. Enter Amount: Provide the amount of ETH you want to withdraw.

            
3. Withdraw ETH: Click the "Withdraw ETH" button to transfer ETH from the specified wallet.

        

        

            
10. Deposit ERC20

            
1. Enter Token ID: Input the ID of the wallet to which you want to deposit ERC20 tokens.

            
2. Enter ERC20 Token Address: Provide the contract address of the ERC20 token.

            
3. Enter Amount: Provide the amount of ERC20 tokens you want to deposit.

            
4. Deposit ERC20: Click the "Deposit ERC20" button to transfer tokens to the specified wallet.

        

        

            
11. Withdraw ERC20

            
1. Enter Token ID: Input the ID of the wallet from which you want to withdraw ERC20 tokens.

            
2. Enter ERC20 Token Address: Provide the contract address of the ERC20 token.

            
3. Enter Amount: Provide the amount of ERC20 tokens you want to withdraw.

            
4. Withdraw ERC20: Click the "Withdraw ERC20" button to transfer tokens from the specified wallet.

        

        

            
12. Deposit ERC721

            
1. Enter Token ID: Input the ID of the wallet to which you want to deposit an ERC721 NFT.

            
2. Enter ERC721 NFT Address: Provide the contract address of the ERC721 NFT.

            
3. Enter NFT ID: Provide the ID of the ERC721 NFT you want to deposit.

            
4. Deposit ERC721: Click the "Deposit ERC721" button to transfer the NFT to the specified wallet.

        

        

            
13. Withdraw ERC721

            
1. Enter Token ID: Input the ID of the wallet from which you want to withdraw an ERC721 NFT.

            
2. Enter ERC721 NFT Address: Provide the contract address of the ERC721 NFT.

            
3. Enter NFT ID: Provide the ID of the ERC721 NFT you want to withdraw.

            
4. Withdraw ERC721: Click the "Withdraw ERC721" button to transfer the NFT from the specified wallet.

        

        

            
14. Change Label

            
1. Enter Token ID: Input the ID of the token for which you want to change the label.

            
2. Enter New Label: Provide the new label for the token.

            
3. Change Label: Click the "Change Label" button to update the label.

        

        

            
15. Transfer Token

            
1. Enter Token ID: Input the ID of the token you want to transfer.

            
2. Enter Recipient Address: Provide the Ethereum address of the recipient.

            
3. Transfer Token: Click the "Transfer Token" button to initiate the transfer.
